# Equation Genius

Equation Genius is an iOS equation game that helps players improve mental math by solving auto-generated algebra-style equations.

## Features

- Auto-generated equation puzzles
- Challenge Mode
- Play Mode
- Multiplayer
- AI opponents
- Game Center leaderboard
